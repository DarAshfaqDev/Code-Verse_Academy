# CodeVerse Academy

Modern educational platform inspired by W3Schools, redesigned as a premium learning dashboard for Web Development, Data Science, AI/ML, Programming, Interview Preparation and Projects.

## Stack

- Frontend: Next.js, React, TypeScript, Tailwind CSS
- UI: Framer Motion, Lucide icons, Recharts, Monaco Editor
- Backend: Node.js, Express
- Auth-ready: JWT scaffold, compatible with Clerk or Firebase Auth
- Database-ready: MongoDB/PostgreSQL architecture notes and environment placeholders

## Pages

- Home
- Courses
- Tutorial page
- Playground
- Dashboard
- Login and Register
- Profile
- Admin panel
- Blog
- Community
- Certifications
- Projects showcase
- Data Science
- AI/ML
- Practice

## Features

- Responsive sticky sidebar navigation
- Dark and light mode
- Animated hero and code snippets
- Course cards with chapters, progress and certificate indicators
- Monaco split-screen playground with live HTML output and simulated Python/SQL modes
- Student dashboard with XP, streaks, badges, charts, notes and recently viewed lessons
- Practice challenge queue with MCQs, coding, SQL and dataset-oriented drills
- Admin panel for courses, users, quizzes, certificates and analytics
- Realistic dummy content for courses, quizzes, projects and dashboard metrics
- Express API scaffold for courses, dashboard and JWT login

## Install

```bash
npm install
```

## Run Frontend

```bash
npm run dev
```

## Run Backend

```bash
npm run server
```

## Run Both

```bash
npm run dev:full
```

Frontend runs on `http://localhost:3000`. API runs on `http://localhost:4000`.
